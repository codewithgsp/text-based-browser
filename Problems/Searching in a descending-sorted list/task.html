Searching in a descending-sorted list Open on JetBrains Academy

Write a program that finds the left-most position of a target element in a descending-sorted list.

Input: the first line contains a descending-sorted list of numbers separated by spaces. The second line contains a target element.

Output: the left-most position of the target element in the list or $-1$ if the target element is not found.

Note that this problem can be easily solved using a for loop, but we ask you to implement the algorithm so that you understand it better!

You need to modify the behavior of the algorithm in the case when the middle element equals target. Instead of returning the index, you need to call the method again.


Sample Input:
5 5 4 4 4 4 3 2 1
4
Sample Output:
2


Memory limit: 512 MB
Time limit: 3 seconds
